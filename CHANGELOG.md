# Changelog

## 0.1.0

- Initial release of `Sentinels Main Front UI`.
- Added reusable design tokens (colors, typography, spacing, radius, elevation).
- Added `SentinelsTheme.light` and `SentinelsTheme.dark`.
- Added `SentinelsAdminTheme` extension for admin surfaces.
- Added `SentinelsThemeController` for mode/seed management.
- Added documentation and runnable example.
