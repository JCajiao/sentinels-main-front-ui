# Design Tokens

## Color System

Primary source is Morvin, with Upzet-driven semantic balancing.

| Token | Hex | Purpose |
|---|---|---|
| `primary` | `#525CE5` | Brand primary actions |
| `primaryContainer` | `#E0E3FF` | Soft primary surfaces |
| `secondary` | `#0BB197` | Secondary action/accent |
| `success` | `#23C58F` | Success state |
| `warning` | `#F0B24B` | Warning state |
| `danger` | `#F14E4E` | Error/destructive state |
| `info` | `#5BA4E5` | Informational state |
| `backgroundLight` | `#F1F5F7` | App background (light) |
| `surfaceLight` | `#FFFFFF` | Cards/surfaces (light) |
| `borderLight` | `#E8ECF1` | Light border/divider |
| `backgroundDark` | `#1D222E` | App background (dark) |
| `surfaceDark` | `#252B3B` | Cards/surfaces (dark) |
| `borderDark` | `#37404A` | Dark border/divider |

## Typography

- Family: `Inter` (Morvin base)
- Strategy: global text theme through `GoogleFonts.interTextTheme`
- Direction: default `LTR`
